# Project: Text Exploratory Data Analysis

**Dataset:** [20 Newsgroups](http://qwone.com/~jason/20Newsgroups/) (via Scikit-Learn)
**Type:** Natural Language Processing (NLP)

## Overview
This project demonstrates how to turn raw text into actionable insights. Unlike tabular data, text requires significant preprocessing before it can be analyzed mathematically. We will explore the "20 Newsgroups" dataset, a standard benchmark for text classification and clustering.

## Goal
To implement a complete end-to-end NLP EDA pipeline:
1.  **Text Cleaning:** Tokenization, Stopword removal, Lemmatization.
2.  **Statistical Analysis:** Word frequencies (Zipf's Law), N-grams.
3.  **Vectorization:** TF-IDF (Term Frequency - Inverse Document Frequency).
4.  **Topic Modeling:** Using Latent Dirichlet Allocation (LDA) to discover hidden thematic structures.

## Notebooks
1.  `01_text_cleaning_and_frequency.ipynb`: From raw text to cleaned tokens and visualizations.
2.  `02_sentiment_and_topic_modeling.ipynb`: Uncovering latent topics and clustering documents.

## Key Concepts
- **Bag of Words (BoW)** vs **TF-IDF**
- **Stopwords** and **Lemmatization**
- **N-grams** (Bi-grams, Tri-grams)
- **Latent Dirichlet Allocation (LDA)**
